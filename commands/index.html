# Команды

Примечание

У большинства команд есть небольшая подсказка по использованию, достаточно отправить в чат команду без каких-либо параметров.

# Содержание

  • Общее:
    • !bot - Проверяет бота на функционирвоание.
    • !commands - Ссылка на эту страницу.
  • Антиспам:
    • !antispam - Включает\Выключает все Антиспам-фильтры.
    • !notbot - Разбанивает пользователя забаненного AntiBot фильтром Антиспама.
    • !dlm - Удаляет длинные сообщения.
    • Муты канала:
      • !addmute - Добавляет фразу в список мутов канала.
      • !delmute - Удаляет фразу из списка мутов канала.
      • !clearmute - Очищает список мутов канала.
    • Массовый таймаут:
      • !mt - Проходит по сообщениям в чате и удаляет сообщения с указанным текстом.
  • AutoMod твича:
    • !addbt - Добавляет указанный текст в список блокируемых слов AutoMod'а.
      НЕ Для БАНВОРДОВ.
    • !delbt - Удаляет указанный текст из списка блокируемых слов AutoMod'а.
    • !ss - Временно добавляет фразу в список блокируемых слов AutoMod'а.
    • !ssc - Удаляет фразы из AutoMod'а, добавленные с помощью команды !ss.
  • Взаимодействие с чатом:
    • !ecmd - Добавляет "команду", которой можно пропиарить соцсети или другие ссылки.
    • !elinks - Показывает команды добавленные через !ecmd.
    • !rm - Повторяет сообщение указанное число раз, от 1 до 20.
    • !ra - Повторяет сообщение с анонсом указанное число раз, от 1 до 20.
  • Таймеры:
    • !etimer - Добавляет таймер канала.
    • !etimers - Показывает список таймеров канала.
  • Развлечения:
    • !song - Показывает текущий трек транслируемый в статусе vk.
    • !giveaway - Запускает розыгрыш в чате по ключевому слову.
    • !snejok - Случайным образом кидает снежок в одного из пользователей чата.

Примечание

Уровень доступа к командам - moderator, если не указано иное.

# Антиспам

# !antispam

Включает\Выключает все Антиспам-фильтры.

Использование:

  • !antispam [on|off]

Встроенные фильтры:

  • AntiBot - от спам-ботов, по умолчанию ВКЛ
  • BanWord - от банвордов, по умолчанию ВКЛ
  • SpamWord - от некоторого спама и рекламы, по умолчанию ВКЛ
  • PhoneNumber - от номеров телефонов, по умолчанию ВКЛ
  • CardNumber - от номеров карт, по умолчанию ВКЛ
  • IPAddress - от IP адресов, по умолчанию ВЫКЛ
  • DLM фильтр, по умолчанию ВыКЛ

Если необходимо включить или отключить какой-то отдельный фильтр, то пишите мне в лсopen in new window.

# !notbot

Разбанивает пользователя забаненного AntiBot фильтром Антиспама.

Использование:

  • !notbot [никнейм]

# !dlm

Антиспам-фильтр для удаления длинных сообщений. По умолчанию ВЫКЛ.

Использование:

  • !dlm [on|off|число]

число - ограничение длины сообщения, значение в диапазоне от 150 до 500 символов
Выдает таймаут на 30 секунд, при повторных срабатываниях - 120, 240

# !addmute

Муты канала.

Добавляет фразу в список мутов канала.
Автоматически отстраняет пользователя если его сообщение содержит указанный текст.

  • Alias команды - !m
  • Продолжительность по умолчанию - 5 минут
  • Максимальная продолжительность - 2 недели

Использование:

  • !addmute [продолжительность] [текст]

Примечание: первое слово в тексте, начинающееся с числа, будет обработано как продолжительность таймаута.

# !delmute

Удаляет фразу из списка мутов канала.

Использование:

  • !delmute [текст]

# !clearmute

Очищает список мутов канала.

  • Alias команды - !stop

# !mt

Массовый таймаут.

Проходит по последним отправленным сообщениям в чате.
Если сообщение содержит указанный текст, то пользователь будет отстранен на указанное количество времени.

  • Alias команды - !masstimeout
  • Продолжительность по умолчанию - 5 минут
  • Максимальная продолжительность - 2 недели

Использование:

  • !mt [продолжительность] [текст]

Примечание: первое слово в тексте, начинающееся с числа, будет обработано как продолжительность таймаута.

# AutoMod твича

# !addbt

Добавляет указанный текст в список блокируемых слов AutoMod'а.

Использование:

  • !addbt [текст]

Примичание: НЕ Добавляйте БАНВОРДЫ через эту команду

# !delbt

Удаляет указанный текст из списка блокируемых слов AutoMod'а.

Использование:

  • !delbt [текст]

Примичание: на данный момент удаляет только текст добавленный через бота

# !ss

Временно добавляет фразу в список блокируемых слов AutoMod'а.

  • Фраза будет автоматически удалена из AutoMod'а по истечении времени
  • Продолжительность по умолчанию - 10 минут
  • Максимальная продолжительность - 6 часов

Использование:

  • !ss [продолжительность] [текст]

# !ssc

Удаляет фразы из AutoMod'а, добавленные с помощью команды !ss

# Взаимодействие с чатом

# !ecmd

Добавляет пользовательскую команду, которой можно пропиарить соцсети или другие ссылки.
Модераторы могут вызвать пользовательскую команду с указанием количества повторений, от 1 до 20.

Использование:

  • !ecmd [add|del|edit] [названиеКоманды] [текст]

Добавить команду:

  • !ecmd add hi привет - команда hi с текстом привет

Изменить текст команды:

  • !ecmd edit hi hello - изменить текст на hello

Удалить команду:

  • !ecmd del hi

Вызвать команду:

  • !hi - Вызвать команду однократно, уровень доступа user
  • !hi 3 - Вызвать команду три раза, уровень доступа moderator

Добавление алиаса команды:

  • !ecmd [alias add|alias del] [названиеКоманды] [текст]

  • Примеры:

    • !ecmd alias add тг tg - Добавить команде тг алиас tg
    • !ecmd alias del тг tg - Удалить у команды тг алиас tg

# !elinks

Показывает команды добавленные через !ecmd.

  • Уровень доступа - user

# !rm

Повторяет сообщение указанное число раз, от 1 до 20.
По умолчанию повторяет 5 раз.

Использование:

  • !rm [число] [текст]

# !ra

Повторяет сообщение с анонсом указанное число раз, от 1 до 20.
По умолчанию повторяет 5 раз.

Использование:

  • !ra [число] [текст]

# Таймеры

# !etimer

Добавляет таймер канала. Таймер периодически отправляет сообщение с указанным текстом в чат.

  • Таймеры срабатывают только во время стрима
  • Интервал по умолчанию - 5 минут
  • Количество отправляемых сообщений по умолчанию - 1
  • Максимальное количество таймеров на канал - 3

Использование:

  • !etimer [add|del|edit] [названиеТаймера] [текст] -i=[продолжительность] -c=[количество] -a

    • Необязательные параметры:

    -interval, -i - продолжительность указывается в следующем формате
    -count, -c - количество отправляемых сообщений при срабатывании таймера, от 1 до 5
    -announce, -a - использовать режим анонса при отрпавке сообщения

Добавить таймер:

  • Таймер с параметрами по умолчанию:
    !etimer add test text
  • Таймер с параметрами по умолчанию, с режимом анонса:
    !etimer add test text -a
  • Таймер с интервалом в 60 секунд, с режимом анонса:
    !etimer add test text -interval=1m -a
    !etimer add test text -i=60 -a
  • Таймер с интервалом по умолчанию, количеством сообщений 3 и режимом анонса:
    !etimer add test text -announce -c=3

Изменить текст таймера:

  • !etimer edit test newText

Удалить таймер:

  • !etimer del test

# !etimers

Показывает список таймеров канала.

# Развлечения

# !song

Показывает текущий трек, транслируемый в статус пользователя vk.

  • Уровень доступа - user
  • Alias для команды - !трек
  • Откат команды:
    10 секунд для user
    1 секунда для moderator

Привязать страницу пользователя:

  • Уровень доступа - moderator
  • !song addvk vk.com/pagename

Отвязать страницу:

  • Уровень доступа - moderator
  • !song delvk

# !giveaway

Запускает розыгрыш в чате по ключевому слову.

Использование:

  • !giveaway [ключевоеСлово]

Определить победителя:

  • !roll

Завершить розыгрыш:

  • !stopgiveaway

# !snejok

Случайным образом кидает снежок в одного из пользователей чата.
Смайл snejok можно установить тутopen in new window.

  • Уровень доступа - user
  • Alias для команды - !снежок
  • Откат команды - 1 секунда

Использование:

  • !snejok
  • !snejok [имяПользователя]
