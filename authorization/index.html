# Авторизация

Теперь для использования бота необходима авторизация.

Для авторизации перейдите по ссылке:

https://egetbot.com/authopen in new window

После авторизации приложение "Egetbot" появится в списке подключений в настройках твича.

# Отмена авторизации

Для отмены авторизации отключите приложение в настроках твича:

Настроки - Подключения - Egetbot - Отключить
